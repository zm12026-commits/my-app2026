# 💖 ぷりぷりダイアリー

毎日をかわいく、楽しく記録できる「ゆめかわ」な日記アプリです。

## ✨ ぷりぷりポイント

### 1. ぷりぷりなデザイン (Soft & Kawaii UI)
パステルピンクやラベンダーを基調とした、ふんわり丸みのあるデザインです。背景にはキラキラが舞い、毎日開くのが楽しみになります。

### 2. きょうのきぶん選択
5種類の絵文字（😊🙂😐😟😭）から、その日の気分をワンタップで選べます。

### 3. 文字数応援メッセージ
日記を書くとリアルタイムで文字数をカウント。「たくさん書けてえらい！💖」など、あなたを優しく応援します。

### 4. 安心の自動保存
ブラウザの「LocalStorage」を使用して日記を保存します。サーバー不要で、書いた内容はあなただけのブラウザに大切に保管されます。

---

## 📘 使い方
1.  **きろく**: 日付と気分を選んで、今日あったことを入力します。
2.  **ほぞん**: 「まほうの ほぞん」ボタンを押すと、日記が保存されます。
3.  **ふりかえり**: 下の一覧で過去の日記をチェック！
4.  **さくじょ**: ゴミ箱アイコンから日記を消すことができます。

---

## ⚠️ 注意点
- **保存場所**: データはお使いの「ブラウザ」内に保存されます。
- **共有**: デバイス間（PCとスマホなど）でのデータ共有は行われません。
- **消失**: ブラウザのキャッシュを消去するとデータが消える場合があります。
