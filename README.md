# Simple Diary

日々の出来事を手軽に記録できる、軽量でモダンな日記アプリです。

## 🚀 アプリURL
[https://zm12026-commits.github.io/my-app2026/](https://zm12026-commits.github.io/my-app2026/)

---

## 📘 ユーザーマニュアル

### 1. 概要
このアプリは、ブラウザの「LocalStorage」機能を使用して日記を保存します。サーバーへの登録やログインは不要で、すぐに使い始めることができます。

### 2. 主な機能
- **日記の記録**: 日付を選択し、内容を入力して保存。
- **一覧表示**: 記録した内容を日付の新しい順に表示。
- **削除機能**: ゴミ箱アイコンから日記を削除。
- **レスポンシブ**: スマートフォン、タブレット、PCのすべてのデバイスに対応。

### 3. 使い方
1.  **記録**: 日付と内容を入力し、「記録する」ボタンをクリックします。
2.  **閲覧**: ページ下部にカード形式で日記が表示されます。
3.  **削除**: 各日記カードの右上にあるゴミ箱アイコンをクリックします。

### 4. データの保存に関する注意点（重要）
- **保存場所**: データはお使いの「ブラウザ」内に保存されます。
- **非同期**: デバイス間（例：PCとスマホ）でのデータ共有は行われません。
- **データの消失**: ブラウザのキャッシュクリアや履歴消去を行うと、データが消える場合があります。
